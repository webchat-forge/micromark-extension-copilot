# Quadratic Formula

The quadratic formula provides solutions to quadratic equations in the form ax² + bx + c = 0:

\[
x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}
\]

## Discriminant Analysis

The discriminant (Δ = b² - 4ac) determines the nature of roots:

1. **Positive Discriminant**
   When Δ > 0, there are two real roots:
   \[
   x_1 = \frac{-b + \sqrt{b^2-4ac}}{2a}, \quad x_2 = \frac{-b - \sqrt{b^2-4ac}}{2a}
   \]

2. **Zero Discriminant**
   When Δ = 0, there is one repeated root:
   $$ x = -\frac{b}{2a} $$

3. **Negative Discriminant**
   When Δ < 0, there are two complex roots:
   \[
   x = -\frac{b}{2a} \pm i\frac{\sqrt{|b^2-4ac|}}{2a}
   \]
